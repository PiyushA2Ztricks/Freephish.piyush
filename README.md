## Usage
Clone it by
```bash
git clone https://github.com/anubhavanonymous/Freephish
```
Now, Navigate to Freephish Directory
```bash
cd Freephish
```
Install the requirements by
```bash
bash setup.sh
```
Run it 
```bash
bash freephish.sh
```
